# Dreamworld Cafe Database

A taste of data, brewed with creativity and code.

---

## Overview
Dreamworld Cafe is a relational SQL database system designed to manage cafe operations:
customers, products, orders, staff, and reviews.
This repository will contain the schema, sample data, example queries, and an ERD image.

---

## Repo Structure
Dreamworld-Cafe-DB/
  sql/
    create_tables.sql        # table creation scripts
    insert_sample_data.sql   # sample data for testing
    sample_queries.sql       # example queries and reports
  docs/
    ERD.png                  # ERD image(s) exported from your design tool
  .gitignore
  README.md
  LICENSE

---

## Planned Contents
1. Database Schema (create_tables.sql) - Customers, Products, Orders, Order_Items, Staff, Reviews
2. ERD Diagram (docs/ERD.png) - visual relationships between tables
3. Sample Data (insert_sample_data.sql) - realistic rows to test queries
4. Useful Queries (sample_queries.sql) - sales reports, top products, customer history
5. Optional: small demo app to connect to the DB (Flask or Node)
